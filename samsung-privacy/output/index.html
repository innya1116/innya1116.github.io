HTML 작성규칙

  • 모든 html 코드는 소문자로 작성한다.
  • 들여쓰기는 공백 2칸으로 사용한다.
  • 가독성을 고려하여 빈 줄을 추가할 수 있다. 단 연속적으로 1줄을 초과하지 않는다.
  • 자동닫기 요소의 마지막 /는 생략(예. img, input, br 등)
  • 페이지 이동을 위한 링크 버튼을 제외하고 모든 버튼은 기본적으로 button 태그 사용.
    (button을 a로 사용해야 할 경우를 고려하여, element를 변경해도 스타일은 유지할 수 있도록 마크업)
  • 반응형으로 구현 및 웹표준, 웹접근성을 준수하여 마크업

주석 표기

영역주석

<!-- 영역명 -->
<div>
  <p>시작과 끝이 있고, 마크업의 영역을 표시한다.</p>
</div>
<!--// 영역명 -->

<div class="area">
  <p>class의 범위를 표기하는 주석, 영역이 끝나는 마지막 닫기 태그 옆에 표시한다.</p>
</div><!--// .area -->

설명주석

<!-- 설명내용 -->
<p>시작과 끝이 없고, 설명할 내용을 주석으로 표기한다.</p>

다국어 언어 설정

  • open 대상 국가: 87개국
  • 다국어 작업을 위해 언어별로 lang 선언하여 사용
  • lang 값이 변경되면 기본 폰트로 선언된 SamsungOne 폰트를 각 언어별 기본 폰트로 변경되도록 작업
  • rtl 언어 사용 국가의 경우, html에 dir="rtl"을 선언

Text to Speech

  • Add a button to read titles and some text in English
  • Activated only when html language is en (lang="en")
  • After clicking button.tts-btn, the disabled attribute is added during the time when text is output as voice


Type 1: Use data-target-id(target only one id)

- Read the text of the target ID


Text to Speech Test1

<p id="tts-type1">Text to Speech Test1</p>
<button class="tts-btn" data-target-id="tts-type1"><span class="hidden">Click to start listening</span></button>

Type 2: Use data-target-id(target multiple id)

- Read it in order if there are multiple ids

Title Text

Description Text

<div class="page-top">
  <h1 class="page-top__title">
    <strong id="ttsPi41">Title Text</strong>

  </h1>

</div>

Layer

Use type 1

페이지 내에 팝업 코드를 넣어주고 div.popup 영역의 id 값을 버튼의 data-popup-id 값에 넣어줌

layer popup open 
<a href="#" class="cta cta--outlined" data-popup-id="layer-id">layer popup open</a>

Use type 2

페이지 내에 팝업 코드를 넣어주고 div.popup 영역의 id 값을 버튼의 onclick에 mvJs.popup.open('#{id값}') 선언

layer popup open 
<a href="#" class="cta cta--outlined" onclick="mvJs.popup.open('#layer-id')">layer popup open</a>

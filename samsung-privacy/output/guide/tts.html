Text to Speech

  • 영문어권에서 타이틀 및 일부 텍스트를 읽어주는 버튼 추가
  • html language가 en인 경우에만 활성화 됨(lang="en")
  • button.tts-btn 클릭 후 텍스트가 음성으로 출력되는 시간 동안 disabled 속성 추가됨


Type 1: Use data-target-id(id 한개 사용)

- 타겟 id의 텍스트를 읽어줌


Text to Speech Test1

<p id="tts-type1">Text to Speech Test1</p>
<button class="tts-btn" data-target-id="tts-type1"><span class="hidden">Click to start listening</span></button>

Type 2: Use data-target-id(id 여러개 사용)

- 여러개의 id를 사용할 경우 순서대로 읽어줌

Title Text

Description Text

<div class="page-top">
  <h1 class="page-top__title">
    <strong id="ttsPi41">Title Text</strong>
    <button class="tts-btn" data-target-id="ttsPi41 ttsPi42"><span class="hidden">Click to start listening</span></button>
  </h1>
  <p class="page-top__description" id="ttsPi42">Description Text</p>
</div>

Type 3: data-label

- data-label의 텍스트를 읽어줌

<button class="tts-btn" data-label="Text to Speech Test3"><span class="hidden">Click to start listening</span></button>

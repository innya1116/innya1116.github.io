HTML Creation Rule

  • All html codes are written in lowercase.
  • Indentation is used with two spaces.
  • Empty lines can be added in consideration of readability. However, it does not exceed one line in a row.
  • The last / of the auto-closing element is omitted (e.g. img, input, br, etc.)
  • All buttons, except the link button for page movement, use the button tag by default.
    (In consideration of the case where button should be used as "a", markup so that the style can be maintained even if the element is changed)
  • Markup in compliance with responsive implementation, web standards, and web accessibility

Input

입력 전

<div class="wrapInp">
	<input type="text" id="inputId">
	<label for="inputId">이름</label>
</div>

입력 중 & 입력된 정보 수정 시

<div class="wrapInp on"><!-- 입력데이터가 없는 경우 on 클래스 삭제 -->
	<input type="text" id="inputId2" value="홍길동">
	<label for="inputId2">이름</label>
</div>

입력된 정보 출력 시

<div class="wrapInp disabled"><!-- 수정할 수 없는 상태의 input ( readonly or disabled )-->
	<input type="text" id="inputId4" value="홍길동" readonly>
	<label for="inputId4">이름</label>
</div>

주소검색

	<div class="wrapInp zipCode"><!-- 우편번호 검색 팝업에서 받은 정보가 입력되면 class="on" 추가 -->

		<label for="idZipCode">우편번호</label>
		<button type="button" class="btnSearch">주소검색</button>
	</div>
	<div class="wrapInp addr"><!-- 우편번호 검색 팝업에서 받은 정보가 입력되면 class="on" 추가 -->

		<label for="idAddress1" class="icoAddr">주소</label>
	</div>
	<div class="wrapInp addr">
		<input type="text" id="idAddress2">
		<label for="idAddress2" class="icoAddr">상세주소</label>
	</div>
	<p class="inpErr">유효성 체크 오류 메시지</p>
</div>

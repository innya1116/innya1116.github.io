Usage

  • 지원 브라우저 : Edge, Chrome, firefox, safari
  • 페이지 로드 이후 .select-wrap>select 가 존재하는 경우 디자인 셀렉트용 div가 생성되며 기본 셀렉트는 display:none로 변경됨
  • 셀렉트에 이벤트를 바인딩 하는 경우는 select , change 두 가지로 바인딩 할 수 있도록 설정되어 있다
  • select option 항목이 동적으로 생성/삭제되는 경우 select에 id 값을 추가하고 해당 id로 update 
    mvJs.selectmenu.update('#{select에 부여한 id}');
  • Select Data Option 값 설정 
    data-options='{
  "maxHeight": "250", // 셀렉트 메뉴 최대 높이 지정
  "direction": "down", // 셀렉트 메뉴 열리는 방향 설정 up/down 디폴트값 down
  "msg": "Default select example" // 셀렉트 메시지 설정
          // 옵션이 선택되어 있어야 하는 경우에는 data-msg 옵션 사용하지 않고 select option에 selected 속성 추가
}'

체크박스 리스트 기능 설명

  • 디폴트 3개년 전체 선택 활성화(상단 표기 : 2018 ~ 2020)
  • 최대 선택 3개년 / 최소 1개년
  • 연결되는 연도 선택 시 물결표시로 구분(상단표기 : 20XX ~ 20XX)
    => 예) 2019 ~ 2020 / 2018 ~ 2019
  • 1개년도씩 선택 가능(상단 표기 : 20XX)
  • 2개년도 이상 (연결 되지 않는 연도) 선택 시 콤마표시로 구분(상단 표기 : 20XX, 20XX)
  • 2개년도 이상 (중복타입) 선택 시 물결, 콤마 사용(상단 표기 : 20XX ~ 20XX, 20XX / 20XX, 20XX ~ 20XX)

Layer Popup

  • 팝업을 호출하고자 하는 버튼에 data-popup-id="팝업ID" 추가
  • 호출되는 div.popup에 id="팝업ID" 동일하게 매칭
  • div.popup의 aria-labelledby="팝업타이틀ID" / h2.popup-title의 id="팝업타이틀ID" 값 매칭하여 사용
  • js로 사용할 경우 
    <!-- popup open -->
mvJs.popup.open('#팝업ID');

<!-- popup close -->
mvJs.popup.close('#팝업ID');
